# 2D像素挖掘游戏 - 实现计划

## 项目概述
创建一个向下挖掘的2D像素方块游戏，玩家控制角色不断向下挖掘，收集各种矿石，越深处宝藏越珍贵，可以升级装备提升挖掘能力。

## 技术选型
- **技术栈**: 纯 HTML5 Canvas + JavaScript（无需构建工具，可直接浏览器运行）
- **视觉风格**: 纯代码绘制像素方块，无需外部资源
- **复杂度**: 中等版 - 包含多种矿石、装备升级系统、商店

---

## 游戏核心机制

### 1. 地图系统
- 程序化生成的2D方块地图
- 地图深度：100层
- 每层宽度：20格
- 方块类型随深度变化，越深越稀有

### 2. 方块类型（从浅到深）
| 深度范围 | 方块类型 | 硬度 | 价值 |
|---------|---------|-----|------|
| 0-10层 | 泥土 | 1 | 1金币 |
| 10-25层 | 石头 | 2 | 2金币 |
| 25-40层 | 铜矿 | 3 | 5金币 |
| 40-55层 | 银矿 | 4 | 10金币 |
| 55-70层 | 金矿 | 5 | 20金币 |
| 70-85层 | 钻石 | 6 | 50金币 |
| 85-100层 | 稀有宝石 | 7 | 100金币 |

### 3. 装备系统
- **镐子等级**: 木镐 → 石镐 → 铜镐 → 银镐 → 金镐 → 钻石镐
- 每级提升挖掘速度和可挖掘硬度上限
- 升级需要金币 + 特定矿石

### 4. 玩家属性
- 当前金币
- 当前深度
- 挖掘速度
- 背包容量（矿石存储）

### 5. 操作方式
- 键盘方向键/WASD：移动
- 空格键：挖掘当前方向方块
- E键：打开商店/升级界面

## 核心代码设计

### 方块类结构
```javascript
class Block {
  constructor(type, x, y) {
    this.type = type;        // 方块类型
    this.x = x;              // 网格x坐标
    this.y = y;              // 网格y坐标（深度）
    this.hardness = 0;       // 硬度
    this.value = 0;          // 价值
    this.color = '#000';     // 颜色
  }
}
```

### 玩家类结构
```javascript
class Player {
  constructor(x, y) {
    this.x = x;
    this.y = y;
    this.gold = 0;

  move(dx, dy) {}
```
